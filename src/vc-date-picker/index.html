Material 2 Date Picker Web Component

This example demonstrates the usage of the vc-date-picker custom element that implements a Material Design 2 style date picker.

Basic Usage

Selected value: none 
<vc-date-picker id="basic-date-picker" label="Select a date"></vc-date-picker>

With Min and Max Date

Selected value: none 
<vc-date-picker 
  id="range-date-picker" 
  label="Select a date within range" 
  min="2023-01-01" 
  max="2023-12-31">
</vc-date-picker>

With Initial Value

Selected value: 2023-07-15 
<vc-date-picker 
  id="initial-date-picker" 
  label="With initial value" 
  value="2023-07-15">
</vc-date-picker>

Disabled Date Picker

<vc-date-picker 
  id="disabled-date-picker" 
  label="Disabled date picker" 
  value="2023-06-10" 
  disabled>
</vc-date-picker>

Required Date Picker

<vc-date-picker 
  id="required-date-picker" 
  label="Required date picker" 
  required>
</vc-date-picker>
