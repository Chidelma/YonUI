Basic Examples

Switch state: unchecked

Usage Examples

Basic Usage:
<vc-switch label="Default Switch"></vc-switch>
Checked Switch:
<vc-switch label="Checked Switch" checked></vc-switch>
Disabled Switch:
<vc-switch label="Disabled Switch" disabled></vc-switch>
JavaScript Control:
const switchElement = document.querySelector('#my-switch');

// Get current state
console.log('Is checked:', switchElement.checked);

// Set new state
switchElement.checked = true;

// Listen for changes
switchElement.addEventListener('change', (event) => {
  console.log('Switch changed:', event.detail.checked);
});
