vc-badge Web Component Examples

Basic Usage

Default Badge

<vc-badge content="4">
  <button>Notifications</button>
</vc-badge>

Dot Badge

<vc-badge dot>
  <button>Notifications</button>
</vc-badge>

Advanced Features

Max Value

<vc-badge content="42" max="99">
  <button>Regular max</button>
</vc-badge>

<vc-badge content="120" max="99">
  <button>Exceeded max</button>
</vc-badge>

Bordered

<vc-badge content="8" bordered>
  <button>Bordered Badge</button>
</vc-badge>

Offset

<vc-badge content="9" offset-x="10" offset-y="-10">
  <button>Offset Badge</button>
</vc-badge>

Hidden Badge

<vc-badge content="10" model-value="false">
  <button>Hidden Badge</button>
</vc-badge>
